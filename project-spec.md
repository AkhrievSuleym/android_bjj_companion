# Grappling Companion — Спецификация проекта

## Описание идеи

Мобильное приложение для спортсменов-единоборцев (грепплинг, BJJ, борьба), которое объединяет контроль питания, отслеживание веса и дневник тренировок в одном месте. Основной фокус — подготовка к соревнованиям и контроль весовой категории.

---

## Целевая аудитория

- Спортсмены-любители и профессионалы в грепплинге, BJJ, вольной/греко-римской борьбе
- Возраст: 16-40 лет
- Цель: контроль веса перед соревнованиями, отслеживание прогресса, ведение дневника тренировок

---

## Пользовательские сценарии (User Stories)

### US-1: Онбординг и профиль
**Как** новый пользователь,  
**я хочу** создать профиль с моими данными (имя, текущий вес, целевой вес, весовая категория),  
**чтобы** приложение могло рассчитать мои цели по питанию и отслеживать прогресс.

### US-2: Отслеживание веса
**Как** спортсмен,  
**я хочу** записывать свой вес ежедневно и видеть график изменений,  
**чтобы** контролировать подготовку к соревнованиям и видеть прогресс к целевому весу.

### US-3: Дневник питания
**Как** спортсмен,  
**я хочу** искать продукты через поиск и добавлять их в дневник питания,  
**чтобы** отслеживать потребление калорий и БЖУ за день.

### US-4: Календарь тренировок
**Как** спортсмен,  
**я хочу** планировать тренировки в календаре и добавлять заметки о том, что отрабатывали,  
**чтобы** вести структурированный дневник подготовки.

### US-5: Дневник техник
**Как** спортсмен,  
**я хочу** записывать техники и приёмы, которые изучаю, с заметками и категориями,  
**чтобы** не забывать изученное и систематизировать знания.

### US-6: Dashboard (главный экран)
**Как** пользователь,  
**я хочу** видеть на главном экране ключевую информацию: текущий вес, прогресс, калории за сегодня, ближайшую тренировку,  
**чтобы** быстро оценить своё состояние.

## Экраны приложения

### 1. Онбординг (2-3 экрана)
- Приветствие с описанием возможностей приложения
- Ввод данных профиля: имя, текущий вес, целевой вес, весовая категория
- Расчёт дневной нормы калорий (TDEE)

### 2. Dashboard (главный экран)
- Карточка веса: текущий вес, прогресс до цели (в кг и %), мини-график за неделю
- Карточка питания: калории сегодня / цель, прогресс-бар БЖУ
- Карточка ближайшей тренировки: дата, тип, превью заметки
- Быстрые действия: "Добавить приём пищи", "Записать вес"

### 3. Питание
- Дневник питания по дням (переключатель дат)
- Поиск продуктов через USDA API
- Добавление продукта: выбор приёма пищи (завтрак/обед/ужин/перекус), указание граммовки
- Сохранение своих продуктов локально
- Итоги дня: калории, белки, жиры, углеводы

### 4. Календарь тренировок
- Месячный вид с отметками тренировок
- Создание тренировки: дата, тип (грепплинг, спарринги, ОФП, кардио), длительность, заметки
- Просмотр и редактирование тренировки
- История тренировок (список)

### 5. Дневник техник
- Список техник с фильтрацией по категориям
- Категории: тейкдауны, сабмишены, свипы, защиты, контроль
- Карточка техники: название, категория, описание/заметки, дата добавления
- Добавление/редактирование техники

### 6. Прогресс
- График веса за период (неделя/месяц/3 месяца)
- Статистика тренировок: количество за период, распределение по типам
- Достижения (опционально для MVP)

### 7. Настройки
- Редактирование профиля
- Переключение темы (светлая/тёмная)
- Напоминания о взвешивании и приёмах пищи
- Сброс данных

## Функциональные требования

### Работа с API (USDA FoodData Central)
- Поиск продуктов по названию
- Получение информации о КБЖУ продукта
- Обработка ошибок сети (показ сообщения, возможность повторить)
- Кэширование результатов поиска локально

### Локальное хранение данных (Room)
- Профиль пользователя
- История взвешиваний
- Дневник питания (продукты, приёмы пищи, даты)
- Тренировки
- Техники
- Избранные/сохранённые продукты

### UI/UX
- Material Design 3
- Поддержка светлой и тёмной темы
- Простые анимации переходов между экранами
- Анимация графиков при появлении
- Анимация прогресс-баров
- Bottom Navigation для навигации

## Ограничения MVP

### Включено в MVP:
- Онбординг и профиль
- Dashboard
- Дневник питания с поиском через API
- Календарь тренировок
- Дневник техник
- График веса
- Светлая/тёмная тема

### Отложено на будущие версии:
- Напоминания (push-уведомления)
- Экспорт данных
- Синхронизация с облаком
- Достижения и геймификация
- Детальная статистика

---

## Критерии успеха

- Приложение запускается без краша
- Поиск продуктов возвращает результаты из USDA API
- Данные сохраняются между сессиями
- Навигация между экранами работает корректно
- График веса отображает сохранённые данные
- Анимации работают плавно (60 fps)
